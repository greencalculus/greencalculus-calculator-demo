Demo covers three representative markets; the live calculator carries all 31 AIB European countries and 27 Green-e US eGRID subregions.

Why this matters: The residual mix is the emission factor for grid electricity left after all contractual instruments (GOs, RECs, PPAs) are removed from the pool — so it is the correct market-based factor for any consumption you have not backed with certificates. Using the plain grid average instead would double-count the renewables others have already claimed. Residual factors run well above the location-based grid average (Germany ≈0.70 vs a ~0.38 grid average), which is the cost of leaving load uncovered.
